# 数据结构实验

🇺🇸 **English**: [README.md](README.md)

## 项目简介

本仓库为华中科技大学《数据结构》课程项目，使用 C/C++ 实现经典数据结构与基础算法。

项目包含顺序线性表、链表、二叉树、图（邻接表）等数据结构的实现，以及完整实验报告和 Xcode 工程文件。

## 项目内容

- 顺序线性表
- 链表
- 二叉树
- 图（邻接表）
- 文件读写
- 控制台交互程序
