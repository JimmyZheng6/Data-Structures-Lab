本实验所有程序均在 macOS 环境下使用 Xcode 开发、调试和测试完成。

由于课程要求提交可独立于 IDE 运行的目标程序（EXE 文件），因此本人另外使用 MinGW-w64 交叉编译工具链，在 macOS 下生成了对应的 Windows 可执行程序（.exe）。

电子资料目录说明如下：

1. 每个实验文件夹均包含：
   （1）源程序（.c/.cpp/.h）
   （2）Windows 可执行程序（.exe）
   （3）macOS 可执行程序（文件名后标注“(macOS)”）
   （4）Xcode 工程文件（.xcodeproj），方便在 macOS 环境下直接打开工程。

2. 其中：
   - *.exe 为 Windows 平台目标程序，可独立于 IDE 运行；
   - *(macOS) 为 macOS 平台可执行程序，可独立于 Xcode 运行；
   - *.xcodeproj 为 Xcode 工程文件，不影响源程序和目标程序的使用，仅用于工程管理。
