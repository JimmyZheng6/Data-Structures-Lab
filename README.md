## Notes | 说明

- `*.exe` files are Windows executables generated using MinGW-w64.
- macOS executable files can be run directly on macOS.
- `.xcodeproj` files are provided for opening the projects in Xcode.

- `*.exe` 为使用 MinGW-w64 交叉编译生成的 Windows 可执行程序。
- macOS 可执行程序可直接在 macOS 环境下运行。
- `.xcodeproj` 用于在 Xcode 中打开工程。
